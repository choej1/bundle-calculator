# Bundle Calculator — 번들 수익성 계산기

묶음 상품(번들)의 수익성을 빠르게 분석하는 웹 앱입니다.
서버 없이 브라우저에서 동작하며, 모든 데이터는 기기 내 `localStorage`에 저장됩니다.

---

## 주요 기능

- **제품 관리** — 달러/원화 원가 제품 등록, CSV 대량 가져오기/내보내기
- **번들 구성기** — 여러 제품을 묶어 판매가·할인율·비용 항목을 설정하고 실시간으로 수익성 계산
- **번들 비교** — 최대 4개 번들의 매출총이익·공헌이익을 차트로 비교
- **설정**
  - 환율 및 수입비용율(%) 설정
  - VAT(부가세)율 설정
  - 비용 프리셋 관리 (택배비, 플랫폼 수수료 등)

### 계산 기준

| 항목 | 기준 |
|------|------|
| 판매가 | VAT 포함 |
| 원가 | VAT 제외 |
| 달러 원가 환산 | `$원가 × 환율 × (1 + 수입비용율%)` |
| 이익·이익률 | VAT 제외 판매가 기준 |

## CSV 파일 형식 안내

**제품 가져오기** (`제품 관리` 탭 → CSV 가져오기)

| 열 이름 | 필수 | 설명 |
|---------|------|------|
| `name` | ✅ | 제품명 |
| `costKRW` | 원화 원가일 때 | 원화 원가 (숫자) |
| `costUSD` | 달러 원가일 때 | 달러 원가 (숫자) |
| `retailPrice` | | 소비자가 (VAT 포함, 숫자) |
| `memo` | | 메모 |

- `costKRW`와 `costUSD` 중 하나만 입력합니다. 둘 다 있으면 `costUSD`가 우선 적용됩니다.
- 숫자 열에 쉼표(`,`)가 포함된 경우 자동 제거됩니다.
- 첫 번째 행은 헤더로 인식됩니다.

**CSV 예시**

```csv
name,costUSD,retailPrice,memo
프리미엄 세트,12.5,29900,미국 직수입
기본 패키지,,9900,원화 원가 제품
```
